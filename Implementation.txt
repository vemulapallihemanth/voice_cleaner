?? Implementation Details
Database Models
VoiceProfile Model
python
- user (ForeignKey to User)
- name (CharField)
- description (TextField)
- created_at (DateTimeField)
- updated_at (DateTimeField)
- is_active (BooleanField)
Voice Model
python
- profile (ForeignKey to VoiceProfile)
- text (TextField)
- audio_file (FileField)
- duration (FloatField)
- speed (IntegerField)
- pitch (IntegerField)
- volume (FloatField)
- created_at (DateTimeField)
Voice Synthesizer
Technology: pyttsx3 (offline TTS engine)
Features:
?Configurable speech rate (words per minute)
?Adjustable pitch (voice selection)
?Volume control
?Audio file generation
?Duration calculation
Process:
1.User enters text and settings
2.Synthesizer generates speech
3.Audio saved to media/voices/
4.Duration calculated
5.File linked to Voice model
